# C Generic Utility Library & Bitwise Analyzer

This repository contains a system-level C project that demonstrates advanced memory management, custom data structures, and low-level bitwise operations.

## Key Features
* **Dynamic Data Structures:** Implementation of a generic Linked List (`list.h`, `list.c`) from scratch.
* **Generic Algorithms:** A custom implementation of Binary Search (`mybsearch`) using `void*` pointers and function pointers, allowing searching through abstract data types.
* **Bitwise Operations:** Low-level manipulation to parse and analyze binary representations (e.g., finding the maximum consecutive '1' bits in a byte, and applying bitmasks).
* **File I/O:** Reading and writing binary data efficiently.

## Files Included
* `main.c` - The main driver program containing the core logic and bitwise algorithms.
* `list.c` / `list.h` - The Linked List implementation.
* `def.h` - General definitions and node structures.
